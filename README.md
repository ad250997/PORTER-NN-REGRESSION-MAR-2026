# PORTER – Delivery Time Prediction using Neural Networks

Neural network regression model to predict delivery time for intra-city logistics orders on the Porter platform. The project demonstrates an end-to-end deep learning workflow for **tabular regression**, including preprocessing, feature engineering, hyperparameter tuning, and model evaluation.

## Key Learnings

- Building **deep learning models for tabular regression**
- Handling **mixed feature types using embeddings**
- Importance of **feature scaling for neural networks**
- Hyperparameter optimization using **KerasTuner**
- Diagnosing models using **training curves and evaluation metrics**

## Running the Project

Clone the repository and install dependencies:

`pip install -r requirements.txt`

Start the application:

`python run.py`

The Streamlit UI will open automatically at:
http://localhost:8501
